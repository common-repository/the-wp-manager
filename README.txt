== Description ==
 
No #1 manager plugin for WordPress. This plugin is used for increase media upload file size & many more.

Plugin Functionality: 

1) Just activate from backend and it will automatically increase maximum upload file size from 2MB to 75 MB
  
== Get Involved ==


== Installation == 

1) Upload `WP Manager` to the `/wp-content/plugins/` directory
2) Activate the plugin through the 'Plugins' menu in WordPress


Plugin Supports:

=> This plugin includes a function to increase maximum upload file size till 75MB from backend
